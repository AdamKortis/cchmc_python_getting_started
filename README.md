# cchmc_python_getting_started
This is an introductory presentation to using Python for data analytics. It was created for an All Teach, All Learn presentation at Cincinnati Children's Hospital Medical Center. 

## Objectives
1. Understanding Python and its role in data analytics.
2. Setting up Python, software, and tools on your computer.
3. Implementing Python into your workflow for analytics using modules, Excel, and Power BI.
